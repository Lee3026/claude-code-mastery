# Counter App

간단한 카운터 애플리케이션입니다.

## 프로젝트 설명

React와 Tailwind CSS를 사용하여 만든 카운터 컴포넌트입니다. 숫자를 증가, 감소, 초기화할 수 있는 기본적인 기능을 제공합니다.

## 실행 방법

1. `counter.html` 파일을 브라우저에서 엽니다.
2. 또는 파일 탐색기에서 `counter.html`을 더블클릭합니다.

별도의 설치나 빌드 과정이 필요 없습니다!

## 기능

- **증가 (+)**: 카운트를 1씩 증가시킵니다
- **감소 (-)**: 카운트를 1씩 감소시킵니다
- **리셋 (Reset)**: 카운트를 0으로 초기화합니다

## 기술 스택

- React 18 (CDN)
- Tailwind CSS (CDN)
- Babel Standalone
